# Surgery Photo Archive

This directory holds local-only source photos related to Hedy's surgical timeline.

- `Hedy_surgery_photos/` contains the moved original photo folders.
- `Hedy_surgery_photos_manifest.csv` is tracked in git and inventories the photo filenames, sizes, and modified timestamps.
- The `.jpg` and `.jpeg` image files remain excluded from commits and pushes to keep the repository lightweight.

Current manifest summary:

- `20251208_Hedy_DREADD_injection/`: DREADD injection surgery photos.
- `20260331_Hedy_terminal/`: terminal procedure photos.
